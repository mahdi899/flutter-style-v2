Flutter + GitHub Actions + Vercel

آخرین بروزرسانی: 2025-10-18 11:03 UTC

گام‌ها

  1. این پوشه را به ریپوی جدید GitHub با branch main پوش کن.
  2. در Settings → Secrets: VERCEL_TOKEN، VERCEL_ORG_ID، VERCEL_PROJECT_ID را اضافه کن.
  3. در Vercel پروژه را به همین ریپو وصل کن. هر Push → دیپلوی اتومات وب.
  4. برای ساخت APK: Actions → Flutter CI (APK Build) → Run workflow.
